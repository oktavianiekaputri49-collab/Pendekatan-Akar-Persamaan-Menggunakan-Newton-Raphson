# Pendekatan-Akar-Persamaan-Menggunakan-Newton-Raphson
Mini project metode numerik menggunakan Metode Newton-Raphson untuk melakukan pendekatan akar persamaan nonlinear f(x)=x^3 −2x^2 +3x−1.  Dalam project ini dilakukan proses iterasi numerik untuk memperoleh akar persamaan secara bertahap hingga memenuhi batas error relatif yang ditentukan dengan bantuan Microsoft Excel.
